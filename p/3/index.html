AI摘要

AI摘要脚本sm.py 在根目录下,不要移动到其他地方,需要安装python,然后运行

python sm.py

输入slug: slug就是文章的slug字段,在标题的下方,可自定义,不能是中文,前面不要加数字0

输入summary: 输入摘要,可以在浏览器安装kimi阅读助手插件
